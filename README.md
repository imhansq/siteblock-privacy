# siteblock-privacy
siteblock-privacy
> 사이트 차단 타이머(SiteBlock) 개인정보 처리방침
>
> 이 확장 프로그램은 사용자의 개인정보를 별도로 수집하거나 외부 서버로 전송하지 않습니다.
>
> 1. 수집하는 데이터
> - 타이머 설정값(시간, 모드, 알람 여부)
> - 사용자가 입력한 사이트 목록(허용/차단 패턴)
>
> 위 정보는 모두 사용자의 브라우저 내 storage(로컬/동기화 저장소)에만 저장되며, 개발자 또는 제3자의 서버로 전송되지 않습니다.
>
> 2. 데이터 이용 목적
> - 사용자가 설정한 타이머와 목록에 따라 웹사이트를 차단하거나 허용하는 기능 제공
> - 설정 값 유지(브라우저를 재시작해도 기존 설정을 계속 사용하기 위함)
>
> 3. 제3자 제공 및 판매
> - 이 확장 프로그램은 수집된 데이터를 제3자에게 제공하거나 판매하지 않습니다.
>
> 4. 광고 및 추적
> - 광고 네트워크, 분석 도구, 추적 픽셀 등을 사용하지 않습니다.
>
